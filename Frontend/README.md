Netly — Todo Management App
A full-stack todo management application with authentication, theme switching, and task filtering built with React, TypeScript, Redux Toolkit, and a Node.js/Express backend.

Features

Authentication — Email/password login & signup with JWT, plus Google OAuth
Todo Management — Create, read, update, and delete todos
Filtering — Filter todos by status (pending, completed, incomplete) or priority (high, medium, low)
Profile & Settings — Update account details, change password, and upload a profile avatar
Theme Switching — Light and dark mode with persistent preference via Redux
Protected Routes — Auth-gated pages with automatic redirect
Form Validation — Zod schemas with React Hook Form on all forms
